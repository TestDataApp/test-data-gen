## 🌟 Overview

**MockData Builder** is a lightweight, web-based utility designed for QA engineers, developers, and testers to generate realistic dummy data on the fly. Whether you need random emails, phone numbers, or complex JSON structures for API payloads, MockData Builder has you covered.

## ✨ Features

- **📧 Email Generation**: Create random emails with specific domains (Gmail, Outlook, startup.io, etc.) or completely random ones.
- **📱 Phone Number Generation**: Get valid-looking phone numbers for various regions including India (+91), USA (+1), and UK (+44).
- **📦 JSON Payloads**: Quickly generate realistic JSON structures for common testing scenarios:
  - Basic User Info & Auth Payloads
  - E-commerce (Carts, Orders, Checkout)
  - User Address Data
  - API Testing (Pagination, Search/Filter)
  - File Metadata
  - Request Headers (Auth & Content-Type)
- **⚡ Bulk Generation**: Generate up to 100 records in a single click.
- **📋 One-Click Copy**: Easily copy your generated data to the clipboard for instant use.
- **🛠️ Customizable Lengths**: Control the character length of generated strings for email prefixes and JSON values.

## 🚀 Getting Started

No setup or build steps required! Run it locally straight from your browser.

1. Clone the repository or download the source files.
2. Open `index.html` in your favorite web browser.
3. Select your desired data type, configure the count/length, and click **✨ Generate**.

## 💻 Tech Stack

- **HTML5**: Semantic and accessible structure.
- **CSS3**: Clean, responsive, and modern user interface.
- **Vanilla JavaScript**: Fast, dependency-free logic for data generation.

## 💡 How to Use

1. **Select Data Type**: Use the dropdown to choose between Email, Phone, or various JSON schemas.
2. **Set Count**: Enter the number of items you want to generate (between 1 and 100).
3. **Set Length**: Customize the length of email prefixes or JSON string values (between 3 and 100).
4. **Generate & Copy**: Hit "Generate" or press `Enter`, then use the "📋 Copy" button to grab the output.
